# **App Name**: PortfolioPro

## Core Features:

- Hero Section: Full-width hero section with a captivating intro, headline, description, and primary/secondary CTAs. Features a smooth entrance animation on load.
- Project Showcase: Grid layout for displaying projects, with thumbnails, titles, descriptions, tech stack, and links to live demos/source code. Includes a subtle hover effect.
- Contact Form: A minimal contact form to allow users to submit their information and message, which are then saved to Firestore. The LLM tool confirms that there is no inappropriate content.
- About Section: A compelling and minimal about section.
- Skills Section: Categorized skills displayed with minimal icons and a responsive grid layout.

## Style Guidelines:

- Primary color: Dark navy (#0F172A) for a professional and sophisticated look.
- Background color: Light gray (#F8FAFC), almost white, to create a clean and open feel.
- Accent color: Soft blue (#38BDF8) for highlights and interactive elements.
- Body font: 'Inter', sans-serif, used for body text and general UI elements. Note: currently only Google Fonts are supported.
- Headline font: 'Space Grotesk', sans-serif, used for headings to give a modern and slightly techy feel. Note: currently only Google Fonts are supported.
- Use a max-width container of 1200px to center content and maintain consistent spacing.
- Implement subtle scroll animations and hover effects for an engaging user experience.
